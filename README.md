# Task Tracker

A simple task management console application written in Java.  
Users can add new tasks, mark them as completed, and view the list of tasks.

## Features
- Add tasks
- Mark tasks as done
- View all tasks
- Save tasks between sessions (file storage)

## Technologies
- Java 21
- File I/O (`java.io`)
- Collections (`ArrayList`)
